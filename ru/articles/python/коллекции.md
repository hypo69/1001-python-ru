## 5. –ú–Ω–æ–∂–µ—Å—Ç–≤–∞ ‚Äî `set`

## 8. deque ‚Äî –¥–≤—É—Å—Ç–æ—Ä–æ–Ω–Ω—è—è –æ—á–µ—Ä–µ–¥—å

```

## 9. Counter ‚Äî —Å—á—ë—Ç—á–∏–∫ —ç–ª–µ–º–µ–Ω—Ç–æ–≤

–ü–æ–¥—Å—á–∏—Ç—ã–≤–∞–µ—Ç —á–∞—Å—Ç–æ—Ç—É —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –≤ –∏—Ç–µ—Ä–∏—Ä—É–µ–º–æ–º –æ–±—ä–µ–∫—Ç–µ. –£–¥–æ–±–µ–Ω –¥–ª—è —Å—Ç–∞—Ç–∏—Å—Ç–∏–∫–∏, –∞–Ω–∞–ª–∏–∑–∞.

```

> üí° –ü–æ–ª–µ–∑–µ–Ω –¥–ª—è –∞–Ω–∞–ª–∏–∑–∞ —Ç–µ–∫—Å—Ç–æ–≤, –ª–æ–≥–æ–≤, –≥–æ–ª–æ—Å–æ–≤–∞–Ω–∏–π ‚Äî –≤–µ–∑–¥–µ, –≥–¥–µ –Ω—É–∂–Ω–æ –ø–æ—Å—á–∏—Ç–∞—Ç—å, ¬´—á—Ç–æ –≤—Å—Ç—Ä–µ—á–∞–µ—Ç—Å—è —á–∞—â–µ¬ª.

### 2. –°–∫–æ—Ä–æ—Å—Ç—å –¥–æ—Å—Ç—É–ø–∞: `list` vs `tuple` vs `array.array`

### 7. –°–∫–æ—Ä–æ—Å—Ç—å –¥–æ–±–∞–≤–ª–µ–Ω–∏—è: `list.append` vs `deque.append` vs `deque.appendleft`
