## Использование руководства

### Для планирования миграции:

1. Прочитайте [INDEX](00-INDEX.md) для общего понимания
2. Изучите [Главу 1](01-inventory.md) - проведите инвентаризацию
3. Оцените сложность и риски

### Для тестовой миграции:

1. Следуйте главам 2-7 последовательно
2. Используйте готовые скрипты
3. Документируйте проблемы

### Для production миграции:

1. Изучите ВСЕ 13 глав
2. Подготовьте план отката ([Глава 13](13-cutover-finalization.md))
3. Настройте мониторинг ([Глава 12](12-monitoring-maintenance.md))
4. Выполняйте по чек-листам
