CHIEF : ================= Difficulty: 4 ----------------- The game "CHIEF" is a game in which the player acts as a factory manager, planning production. The player defines the quantity of products manufactured of each type, and the computer determines if these values meet the necessary requirements. If not, the player is informed of the incorrect values. The goal of the game is to achieve optimal production, by correctly guessing the quantity of products.

Game rules: 1. The computer guesses three values in the range of 1 to 10: targetA, targetB, and targetC. 2. The player enters their guesses for the values userA, userB, and userC. 3. The computer checks if the entered values match the guessed values. 4. If all three values are guessed correctly, the game ends with a victory. 5. If at least one value does not match, the computer indicates which values were incorrect. 6. The game continues until the player guesses all three values. ----------------- Algorithm: 1. Generate random integers targetA, targetB, and targetC in the range of 1 to 10. 2. Start a loop "as long as not all numbers are guessed": 2.1. Ask the player to enter three integers: userA, userB, and userC. 2.2. Initialize the string message as empty. 2.3. If userA is not equal to targetA, add "A" to message. 2.4. If userB is not equal to targetB, add "B" to message. 2.5. If userC is not equal to targetC, add "C" to message. 2.6. If message is not empty, display the message "WRONG ON " and message. 2.7. Otherwise, display the message "YOU GOT IT!". 3. End of game. ----------------- Flowchart:
