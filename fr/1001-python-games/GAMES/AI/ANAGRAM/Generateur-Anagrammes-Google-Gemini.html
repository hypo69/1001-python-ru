Anagram Generator using Google Gemini

This is a simple code to generate anagrams using the large language model Google Gemini.

Description

The program takes a set of Russian letters as input and tries to find an existing Russian word composed of these letters (using all or part of them).

Anagram Rules

  • Only existing Russian words are used.
  • When searching for anagrams, only Russian letters are considered. Numbers and other symbols are ignored.
  • If it is possible to form several words, one of them is returned.
  • If no word can be formed from the given letters, the message "No anagrams" is returned.

Code Explanation

  • import google.generativeai as genai: Imports the library to interact with Gemini.
  • import re: Imports the library for working with regular expressions (to clean the input).
  • The GoogleGenerativeAI class encapsulates the logic of interaction with the Gemini model.
  • system_instruction: This is the system prompt (instruction) for Gemini, which explains what is expected of it.
  • re.sub(r"[^а-яА-ЯёЁ]", "", q): This line removes from the input string q all characters that are not Russian letters. [^а-яА-ЯёЁ] is a regular expression that means "any character not in the range a-я, А-Я and ёЁ".
  • The check if not q: checks if the string has become empty after removing all non-Cyrillic characters.
  • model.generate_content(q): Sends the request q to the Gemini model.
  • Exception handling try...except prevents the program from crashing in case of errors during API interaction.
