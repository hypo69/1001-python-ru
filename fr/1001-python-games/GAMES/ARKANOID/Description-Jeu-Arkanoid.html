Arkanoid

Description

A classic arcade game where the player controls a horizontal platform at the bottom of the screen, returning a ball that, in turn, bounces off the walls and the platform. The goal of the game is to prevent the ball from falling below the bottom edge of the screen.

Rules

  1. The ball moves on the screen, bouncing off the walls and the platform.
  2. The player moves the platform from left to right.
  3. If the ball hits the platform, it bounces.
  4. If the ball falls below the bottom edge of the screen, the game ends.
