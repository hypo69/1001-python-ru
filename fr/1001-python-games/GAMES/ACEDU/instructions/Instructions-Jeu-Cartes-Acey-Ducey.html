Game Name: Acey-Ducey Card Game

Description

This is a simulation of the Acey-Ducey card game. The player places bets based on the probability that the next card will fall between the two already open cards.

  • Starting Capital: The player starts with 100 dollars.
  • Game Rules:
    1. The computer deals two cards.
    2. The player can decide whether or not to place a bet.
    3. If a bet is placed, a third card is drawn.
    4. If the value of the third card falls between the first two cards, the player wins the bet. Otherwise, the bet is lost.
  • The game ends when the player loses all their capital or ends it manually.

Implementation

Input Data: - User input for: - Initial bet size. - Decision to place a bet or pass.

Output Data: - Message about the player's current capital. - Information about the bet results (win/loss). - Card status at each turn.
