Acey-Ducey Card Game

Description

This is a simulation of the Acey-Ducey card game. The player places bets based on the probability that the next card will fall between the two already open cards.

  • Starting capital: The player starts with 100 dollars.
  • Game rules:
    1. The computer deals two cards.
    2. The player can decide whether or not to place a bet.
    3. If a bet is placed, a third card is drawn.
    4. If the value of the third card falls between the first two cards, the player wins the bet. Otherwise, the bet is lost.
  • The game ends when the player loses all their capital or ends it manually.

How the code works

1. Importing the library


import random
  • random – library for generating random numbers. It is used to shuffle the deck of cards and select random cards.

2. Creating a deck of cards


def create_deck():
    ranks = list(range(2, 15))  # Cards from 2 to 14 (Ace = 14)
    deck = ranks * 4  # 4 suits
    random.shuffle(deck)
    return deck
  • ranks – list of card values from 2 to 14 (Ace = 14).
  • deck – created by multiplying the ranks list by 4 to get 4 suits (spades, hearts, diamonds, clubs).
  • random.shuffle(deck) – shuffles the deck of cards.
  • The function returns the shuffled deck.

3. Displaying the card in a readable format


def card_name(value):
    if value == 11:
        return "Jack"
    elif value == 12:
        return "Queen"
    elif value == 13:
        return "King"
    elif value == 14:
        return "Ace"
    else:
        return str(value)
  • The function takes the numerical value of the card (from 2 to 14) and returns its textual representation.
  • For example, 11 → "Jack", 14 → "Ace".

4. Main game loop


def play_acey_ducey():
    print("Welcome to the Acey Ducey game!")
    print("Rules: You place a bet, guessing if the next card will be between the two already dealt cards.")
    print("If the card is equal to one of the dealt cards or an Ace, you lose.\n")

    money = 100  # Player's starting balance
    deck = create_deck()
  • money – player's starting balance (100 dollars).
  • deck – a deck of cards is created using the create_deck() function.

5. Dealing two cards


while money > 0 and len(deck) >= 3:
    print(f"Your current balance: ${money}")

    # Deal two cards
    card1 = deck.pop()
    card2 = deck.pop()
    while card1 == card2:  # If the cards are identical, draw new ones
        deck.insert(0, card1)
        deck.insert(0, card2)
        card1 = deck.pop()
        card2 = deck.pop()

    print(f"First card: {card_name(card1)}")
    print(f"Second card: {card_name(card2)}")
  • deck.pop() – extracts the last card from the deck.
  • while card1 == card2 – checks if the dealt cards are identical. If so, replaces them at the top of the deck and draws new ones.
  • card_name(card1) – converts the numerical value of the card to text.

6. Determining the range


low_card = min(card1, card2)
high_card = max(card1, card2)
  • low_card – minimum value of the two cards.
  • high_card – maximum value of the two cards.

7. Placing a bet or passing the turn


try:
    bet = int(input(f"Place your bet (from 0 to {money}) or enter '0' to pass the turn: "))
    if bet < 0 or bet > money:
        print("Invalid bet. Please try again.")
        continue
    if bet == 0:
        print("You passed your turn.\n")
        continue  # Pass the turn
except ValueError:
    print("Please enter a number.")
    continue
  • input() – asks the player to place a bet.
  • try-except – handles errors if the player entered a non-numeric value.
  • if bet < 0 or bet > money – checks if the bet is valid.
  • if bet == 0 – if the bet is 0, the player passes the turn.

8. Drawing the next card


next_card = deck.pop()
print(f"Next card: {card_name(next_card)}")
  • next_card – extracts the next card from the deck.
  • card_name(next_card) – converts the card value to textual representation.

9. Checking the result


if next_card == card1 or next_card == card2 or next_card == 14:
    print("You lose !")
    money -= bet
elif low_card < next_card < high_card:
    print("You win !")
    money += bet
else:
    print("You lose !")
    money -= bet
  • if next_card == card1 or next_card == card2 or next_card == 14 – if the next card is equal to one of the dealt cards or if it's an Ace, the player loses.
  • elif low_card < next_card < high_card – if the next card is between the two dealt cards, the player wins.
  • else – in all other cases, the player loses.

10. End of game


if money <= 0:
    print("You have no money left. Game over.")
else:
    print(f"Game over. Your final balance: ${money}")
  • If the player has no money left, the game ends.
  • If the deck of cards is exhausted, the game also ends.

11. Launching the game


if __name__ == "__main__":
    play_acey_ducey()
  • This block launches the game if the file is executed directly (and not imported as a module).
