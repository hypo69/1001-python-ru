The print() function

The print() function in Python is used to display information in the console. It is one of the simplest and most frequently used functions, especially for debugging and displaying data. Let's look at the main aspects of its operation.

Syntax


print(*objects, sep=' ', end='\n', file=sys.stdout, flush=False)

Parameters:

  1. *objects:
    • List of objects to display. You can pass one or more values, separated by commas.
    • Example: 
      
     print('Hello', 'world')
    • Result: Hello world
  2. sep (default ' '):
    • Separator between objects, if multiple are passed.
    • Example: 
      
     print('Hello', 'world', sep=' - ')
    • Result: Hello - world
  3. end (default '\n'):
    • Indicates what will be added to the end of the line. By default, it's a newline.
    • Example: 
      
     print('Hello', end='!')
    • Result: Hello!
  4. file:
    • Stream to which output will be directed (default sys.stdout — standard output).
    • Example: output to a file. 
      
     with open('output.txt', 'w') as f:
         print('Hello, file!', file=f)
  5. flush (default False):
    • If set to True, forces flushing of the output buffer.

Usage examples

Simple string


print('Hello, world!')

Result: Hello, world!

Displaying multiple values


name = 'Anna'
age = 25
print('Name:', name, ', Age:', age)

Result: Nom : Anna , Âge : 25

Customizing the separator


print(1, 2, 3, sep=' -> ')

Result: 1 -> 2 -> 3

Customizing the end of line


for i in range(3):
    print(i, end=' ')

Result: 0 1 2

Using f-strings with variable name display

Starting from Python 3.8, you can use f-strings to display variable values with their names in name=value format. This is useful for debugging, as it allows you to immediately see which variable and which value are displayed.


name = 'Ivan'
age = 30
print(f'{name=}, {age=}')

Result: name='Ivan', age=30

Useful tips for beginners

  1. Debugging code:

    Use print() to check variable values:

    
   x = 10
   y = 20
   print('Sum:', x + y)
  2. Formatted output:

    To display strings with value substitution, it is better to use formatting:

    
   name = 'Ivan'
   age = 30
   print(f'My name is {name}, I am {age} years old.')
  3. Logging:

    In large projects, it is better to use the logging module for output management, but at the beginning, print() helps to quickly display data.
